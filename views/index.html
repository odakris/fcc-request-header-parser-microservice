Request Header Parser Microservice

This is an API that return IP address, the language that the user has set in his browser and the type of software the user is currently on.

Example Usage

[your_url]/api/whoami

Example Output

{ "ipaddress": "::ffff:159.20.14.100", "language": "en-US,en;q=0.5", "software": "Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0" }
